# HR-Analytics-Using-Power-BI
The HR Analytics Dashboard provides a comprehensive overview of employee data, focusing on attrition trends and workforce demographics. Key insights include:  Summary Metrics (top of the dashboard):  Count of Employees: 1,413 employees.  Attrition: 229 employees have left.  Attrition Rate: 16.2% of the workforce.  Average Age: 37 years.
